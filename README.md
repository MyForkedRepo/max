# Installation

```sh
# Let's install the bot.
cd $HOME
git clone github.com/maxcodeteam/max
cd max
chmod +x matador.sh
chmod 777 auto.sh && sed -i -e 's/\r$//' auto.sh
./matador.sh install
./matador.sh 
# Enter a phone number & confirmation code.
```
